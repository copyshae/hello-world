## 完整作業手冊（給 Agent／自己复查）

### A. 開啟與並排

| 步驟 | 做法 |
|------|------|
| 找 Visio | 常見路徑 `C:\Program Files\Microsoft Office\root\Office16\VISIO.EXE` |
| 開檔 | `Start-Process VISIO.EXE -ArgumentList '"完整路徑.vsd"'` |
| 並排 | Cursor 左半屏、Visio 右半屏（可用 Win32 `MoveWindow`） |
| 接上 COM | `[Runtime.InteropServices.Marshal]::GetActiveObject('Visio.Application')` |

### B. 學年

- 字串取代：`113學年` → `115學年`（含標題長句）。
- 右側直行可能是獨立 Shape，文字只有 `113`／`115`：依指定改，勿當班級循環。

### C. 班級對照（建議明日預設）

1. 讀 Excel 兩欄 → `Hashtable`／字典：`舊 → 新`。
2. 走訪頁面所有 Shape（含群組子圖形）。
3. **底色判斷（建議）**  
   - 文字 Shape 或其父層：`FillPattern ≠ 0`（有填滿）才視為「有底塗」。  
   - `FillPattern = 0`（無填滿）→ 底圖教室，**跳過**。  
   - 若父群組有色、子文字無色：以**外層有色**為準（色塊框住文字）。
4. 取出可對照的班級碼：  
   - 連寫：`(?<![0-9])(\d{3})(?![0-9])`  
   - 直書：允許數字間空白／換行，組成三碼後查表，寫回時保留分隔。
5. 只在對照表有該舊碼時替換；一次套用，避免循環誤傷。
6. 存檔；必要時再掃一遍確認無漏／無誤。

### D. 舊法：循環輪替（僅在明確要求時）

- 規則：`1＊＊→2＊＊`、`2＊＊→3＊＊`、`3＊＊→1＊＊`。
- 必須「一次映射」，不可分兩次全替換。
- 直式（有換行分隔）與橫式要分開處理；第二次處理直式時略過已連寫且改過的橫式。
- 明日**預設不要用**；改用 Excel 對照＋底色限定。

### E. 回報格式（建議）

```text
將修改：N 處（有底色且在對照表）
略過：無底色 x／對照表無 y／排除項 z
範例：101→201、特教班 210→310、直書 1⏎0⏎8→2⏎0⏎8
學年：… ；檔名：…
```

- 作法可寫進 `hello-world` 學習日誌並 push。  
- Visio 原檔、含個資／未公開平面圖：**不要**推進公開倉庫。
