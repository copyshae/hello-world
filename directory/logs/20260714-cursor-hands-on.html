20260714 讀取影音檔成可手把手步驟再執行

讀取影音檔內容，整理成可手把手跟做的步驟並執行記錄。稍後若有修改指示，會直接更新本頁。

準備：安裝與漢化

  1. 到 Cursor 官網 下載安裝（Windows／Mac／Linux）。
  2. 用 Google／GitHub／ChatGPT 等帳號登入。
  3. 左上角 Extensions → 搜尋 Chinese → 安裝中文語言包 → 重載，介面變中文。

紅燈常見原因：點開 MCP 詳情看實際指令；mcp-server-chart 要有 Node／npx，TalkToFigma 要有 Bun／bunx，沒裝好會一直紅。

玩法一：一句話把 Excel／CSV 畫成圖表

目的：用 MCP 自動產生柱狀圖、流程圖、詞雲等。

  1. GitHub 開啟 antvis/mcp-server-chart。
  2. 複製專案裡的 MCP 伺服器設定。
  3. Cursor → Settings → MCP → 貼上設定並儲存。
  4. 等左側狀態點變綠色再開始。
  5. 把 Excel／CSV 拖進對話框，說需求，例如：「幫我把這些銷售數據做成柱狀圖」。
  6. Cursor 會呼叫 mcp-server-chart；不滿意再用口令改（顏色、版面等）。
  7. 進階可請它做互動網頁報告、流程圖、詞雲。

Win 參考設定

{
  "mcpServers": {
    "mcp-server-chart": {
      "command": "cmd",
      "args": ["/c", "npx", "-y", "@antv/mcp-server-chart"]
    }
  }
}

玩法二：長文／報告 → 可視化網頁

目的：先看結構清楚的 HTML，再讀原文。

  1. 把文章／研究報告上傳到 Cursor。
  2. 選編程能力強的模型（影片提到如 Claude 系列）。
  3. 請它解析並做成結構清晰的可視化網頁（含摘要、圖表、層次排版）。
  4. 可再要求：「一鍵生成 HTML，我下載後直接打開預覽」。
  5. 不滿意就繼續調排版／配色。
  6. 想要同款風格：把參考網頁連結貼進對話，請它模仿重做並優化體驗。

適用：專案彙報、交接文檔、個人知識整理。

玩法三：本地知識庫（兩步）

目的：優先翻你自己的資料寫報告；不夠再上網補。

步驟 A：建資料夾並用 Cursor 打開

  1. 在電腦建資料夾，例如 MyKnowledgeBase。
  2. 把 PDF／文檔／筆記丟進去。
  3. Cursor → Open Folder 打開這個資料夾。

步驟 B：寫 Rules（告訴 AI 怎麼查）

  1. 先在本地資料夾搜相關內容。
  2. 本地沒結果再在線搜尋補充。
  3. 複雜任務自動呼叫 Sequential Thinking 這類工具，把任務拆小步。

試用範例

對 Cursor 說：「寫一份未成年人網絡保護相關法規研究。」預期：先搜本地 → 拆子任務 → 不夠再上網 → 輸出報告（私密資料留本機）。

玩法四：Cursor + TalkToFigma 做 UI

目的：用嘴巴在 Figma 裡加元件、快速出原型。

  1. 在 Figma 開一個空白設計檔。
  2. 安裝並連上 TalkToFigma MCP／外掛；狀態綠燈再繼續。
  3. 在 Cursor 用自然語言下指令（例如加按鈕、整頁電商首頁）。
  4. Cursor 透過 MCP 在 Figma 建 Frame、導航、間距與示意內容。

MCP 側常見設定（需 Bun）

{
  "mcpServers": {
    "TalkToFigma": {
      "command": "bunx",
      "args": ["cursor-talk-to-figma-mcp@latest"]
    }
  }
}

建議試做順序

順序 玩法 最少要有
1 畫圖表 Node + mcp-server-chart 綠燈
2 長文→網頁 一份 PDF／長文 + Agent 對話
3 本地知識庫 資料夾 + Rules
4 Figma Figma + TalkToFigma 綠燈

先把 MCP 弄成綠燈，再貼資料下指令，最不容易卡關。
