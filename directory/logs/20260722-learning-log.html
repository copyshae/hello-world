20260722 Visio 掃地／教室圖改 115 學年與班級輪替

用 Cursor 搭配本機 Microsoft Visio，批次改「掃地區域／教室分佈」圖：檔名與圖上文字從 113 學年改為 115 學年，並將班級編號做 1→2→3→1 輪替（橫式與直式皆處理）。

一｜今日核心目標

  • 開啟桌面 Visio 檔，並與 Cursor 左右並排，改完立刻在 Visio 看到結果。
  • 檔名與圖內「113 學年」改為「115 學年」。
  • 班級編號輪替：1＊＊→2＊＊、2＊＊→3＊＊、3＊＊→1＊＊。
  • 直書（數字中間有換行）也要同一規則，不能只改橫式。

二｜工作大要

項目做法
檔案桌面 113學年掃地區域分配圖(底定).vsd → 更名 115學年掃地區域分配圖(底定).vsd
開啟方式本機 VISIO.EXE 開啟；PowerShell 將 Cursor 靠左、Visio 靠右並排
學年文字經 Visio COM：標題「113學年」→「115學年」；右邊直行獨立「113」→「115」
橫式班級掃描圖形文字中的三碼班級（如 101／201／301），一次循環替換；約 118 處
直式班級直書常寫成 1⏎0⏎8 這類「數字被換行拆開」；另寫規則只改有分隔的直式，避免把已改過的橫式再轉一次；約 35 處
刻意不改學年標記 115；尺寸標註（如 100mm、119mm）
附帶標籤「美術／體育／舞蹈／特教班」等文字裡的班級碼一併輪替（例如 特教班 210→310）
存檔每次批次改完後 ActiveDocument.Save()，結果直接顯示在右側 Visio
輪替口訣：一年級碼變二年級、二年級變三年級、三年級回到一年級；橫式連寫與直式換行要分開處理，才不會漏改或重複改。

三｜技術做法（摘要）

  1. PowerShell 取得執行中的 Visio：GetActiveObject('Visio.Application')。
  2. 遞迴走訪頁面所有 Shape（含群組子圖形），讀 Characters.Text。
  3. 學年：字串取代 113學年→115學年；右側直行以 Shape ID 精準改數字。
  4. 橫式班級：正則 (?<![0-9])([123])(\d{2})(?![0-9])，首位 1→2、2→3、3→1（同一輪映射，不會互相覆蓋）。
  5. 直式班級：允許數字之間有空白／換行；僅當「非三碼連寫」時才替換，保留原本換行排版。

四｜學到的技術點

  • Cursor 無法內嵌即時預覽 .vsd；要「改完立刻看」最穩是 Visio 本體並排。
  • Visio 直書班級常不是 108 一個字串，而是 1、0、8 中間夾換行；只做一般 Find/Replace 會漏。
  • 三向輪替（1→2→3→1）必須一次映射，不可先全改 1→2 再改 2→3，否則會連改錯。
  • 二次處理直式時，要跳過「已是連寫且改過」的橫式，否則會再轉一輪。
  • 學校圖上常混有尺寸 mm、學年 115，批次規則要設排除條件。
