20260723 導師名單・廁所次數・Visio 掃區大小

今日主線：掃描 PDF 做成班級導師 Excel，對照全年累計與廁所打掃次數；並從 Visio 掃地圖量化各班掃區大小，供新增兩班時騰區參考。（不含桌面圖示排列操作。）

一｜今日核心目標

  • 掃描全能王 PDF → 僅「班級／導師」Excel，並依 101→201→301 排序。
  • 第二份 PDF 對照「全年累計」，依導師姓名併入；近形異字確認同人後補齊。
  • 兩次班級代號換算合併 → 廁所打掃次數，再併入導師表；各年級全年累計由高到低另存。
  • Visio 掃地圖讀出相對面積，產出全校／各年級由大到小兩份表。

二｜工作大要

項目做法與結果
導師名單 OCR 來源 掃描全能王 2026-7-23 09.54.pdf（掃描圖）。RapidOCR＋人工核對姓名。產出僅兩欄：班級、導師（101–111、201–212、301–311）。
全年累計併入 對照 …10.01.pdf。依導師姓名對應全年累計（非依班級）。無同名則空白。
姓名定稿 正確名以導師名單為準：蕭秀綿／鄭雅霙／蔡佩悌 與對照檔近形字為同人並補累計；導師正確寫法為施榮鍾。
廁所打掃次數 兩批班級「首位加一／加二」結果合併計次 → 打掃廁所次數統計.xlsx；再依班級併入導師表「廁所打掃次數」欄。
各年級排序 115學年度班級導師_各年級全年累計排序.xlsx：一／二／三年級分開，全年累計高→低（無資料排該年級最後）。
新增兩班騰區 討論結論：從掃區較大、負擔較重的區域騰出給新班，避免削到本就輕的班。
Visio 掃區大小 讀 114學年掃地區域分配圖1140808.vsd 有填色圖形面積加總。產出全校／各年級「由大至小」兩份 Excel（相對面積＋區塊數＋最大單塊）。
對照原則：全年累計跟「人」走（導師姓名）；廁所次數跟「班」走（班級代號）。新增班騰區則看「區大、負擔重」。

三｜學到的技術點

  • 掃描 PDF 常無文字層，需 OCR；姓名近形（錦／綿、娣／悌、鍾／鐘）要對圖核對，不能只信自動辨識。
  • 合併欄位前先想清楚鍵：累計用姓名、次數用班級，鍵選錯會整表錯位。
  • 班級「首位加 N」可快速模擬升年級後的班碼，再統計次數。
  • Visio Application ProgID 在本機可能異常，改用 InvisibleApp＋檔案複本可避開鎖定；面積用填色 Shape 加總較能代表負責掃區。
  • Excel 正開啟時無法覆寫，可另存新檔名再請使用者關閉後合併。
