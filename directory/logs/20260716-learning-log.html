20260716 Cursor Rules、Excel MCP 投資分析與優先順序表

延續 Cursor 規則／CamScanner／Excel MCP；並用 Excel MCP 與 Python 解析 MY.xlsx，產出獲利前十檔、優先投資順序與圖形分析，寫回桌面「投資」資料夾。

3.3 依已實現獲利：優先投資順位（前 8）

核心／收益成長 科技成長衛星 高報酬但限額（邊境債） 入息／主題衛星

B33197 獲利最高，但新興／邊境債＋澳幣風險高 → 優先序排第 5並設 8%–12% 上限；實際加碼順序以「穩定收益＋拉開黃金相關」為先。

3.4 建議行動：減黃金、輪入獲利實證類型

資金輪動方向：減黃金重複曝險 → B20073／B09460／B38075 等有獲利實證標的；B33197 限額配置。

投資建議摘要：優先 1–3 建立「安聯收益成長 + 科技／機器人」；第 5 的邊境債用想買／想賣價波段做、控上限；第 9–10（印尼債單筆、黃金）暫緩加碼。

五｜學到的要點

  • 勝率高 ≠ 持倉健康；要同時看集中度與未實現損益來源。
  • 優質基金應用帳戶真實已實現獲利排序，再疊配置上限。
  • 圖形能一眼看出「獲利最大的不見得該最先重壓」（例如 B33197）。

建議下一步

  1. 依優先序 1–5 設想買價，分批配置。
  2. 減黃金／原物料重複檔，輪入收益成長與科技衛星。
  3. 把多個 MCP 合併進同一份 mcp.json 並 Refresh。
