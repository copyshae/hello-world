20260717 Cursor 雙機協作與本機檔案系統歸檔

承接雙機工作需求：家用筆電與上班電腦都裝 Cursor，共用檔放 OneDrive、隱私檔留本機 D:。今日以 Cursor Agent 實作「掃描改名 → 桌面／下載／文件／圖片歸檔 → 桌面圖示整理」，並寫下可重複使用的原則。

一｜今日核心目標

  1. 解決「每天搬筆電、插拔線材」：改為雲端共用＋本機隱私分流。
  2. 掃描檔改成可搜尋檔名：YYYY-MM-DD_代表性名稱.副檔名。
  3. 清空雜亂桌面／下載／文件／圖片，系統歸檔到 D:（財務不上雲）。

二｜雙機＋隱私架構（原則）

類型放置位置說明
證件、存摺、財務、家庭 D:\私人\ 不上 OneDrive；僅本機（筆電）處理
學校／家中要共同處理 OneDrive\共同檔案\ 兩邊 Cursor 開同一路徑即可
桌面／下載／文件／圖片歸檔 D:\桌面歸檔\、D:\下載歸檔\、D:\文件歸檔\、D:\圖片歸檔\ 本機大容量整理；原位置留捷徑回指
實作提醒：上班電腦安裝 Cursor + 同一 OneDrive 帳號；共同專案只開 共同檔案，不要把整個 D:\私人 同步上雲。

三｜今日實作大要

1. 資料夾骨架

  • D:\私人\掃描檔、證件合約、財務、家庭
  • OneDrive\共同檔案\學校、家務文件、掃描檔_共用

2. 掃描檔改名與分流

  • 讀取影像／PDF 內容後改名（會考試題、活動門票、證書、退款通知等）。
  • 敏感檔（證件、存摺、申請表）→ D:\私人；可共用掃描 → 共同檔案\掃描檔_共用。
  • 清除「待確認」檔名；桌面招生宣導重複檔刪除（與共同區雜湊相同）。

3. 桌面系統歸檔

  • 財務／家庭 → D:\私人；學校公務、天圓文化、影音、未命名、舊桌面備份 → D:\桌面歸檔。
  • 格式工廠 → D:\工具軟體。
  • 桌面整理為 1_快速存取、2_常用軟體；Explorer 重啟＋對齊格線／自動排列。

4. 下載・文件・圖片搬到 D:

  • 下載：已清空；約 34 GB 分到教科書 ISO、安裝程式、影音、學校、財務等。
  • 文件：學校／天圓／應用程式資料等進 D:\文件歸檔。
  • 圖片：已本機化部分約 3700+ 檔進 D:\圖片歸檔。
未完成：OneDrive「僅雲端」占位檔（文件約 180+、圖片約 1200+）需先在資料夾右鍵「一律保留在此裝置上」下載後，才能繼續搬移。錯誤訊息常見為「雲端檔案提供者未執行」。

四｜學到的技術點

  1. Agent 可當檔案管家：讀圖辨識內容 → 建議檔名 → 搬移／改名；大批量前先預覽較安全。
  2. 隱私分流要先定規則：否則「整理」容易把敏感檔誤丟進雲端共用資料夾。
  3. OneDrive 占位檔 ≠ 本機檔：robocopy／Move-Item 對 ReparsePoint 會失敗；需先 pin／水合。
  4. 桌面圖示：清登錄 IconPos、設 FFlags、重啟 Explorer，比只送 LVM_ARRANGE 更穩。
  5. 命名即搜尋：日期在前＋關鍵詞，之後用檔案總管搜尋比翻資料夾快。

五｜明日可接續

  • OneDrive 恢復後，補搬剩餘雲端圖片／文件。
  • 上班電腦安裝 Cursor，驗證 共同檔案 同步與開啟。
  • 清理 D:\下載歸檔\未命名待整理、空白掃描頁 等低價值檔。
