一｜今日核心目標

  • 桌面產物規則清楚：只留有日期檔；換日刪前一天；可關閉再開。

三｜桌面跑馬燈實作

路徑：MyKnowledgeBase\每日實作\；啟動捷徑指向 launch-daily-marquee.cmd（避開中文路徑打不開）。

項目做法
顯示WinForms 底部置頂窗；左「今日實作」、右上角 X 關閉；Esc 亦可
視覺墨青書院：墨青底＋青瓷分隔＋淡銅短標（見 DESIGN.md）
內容tips.json：短標 2–4 字＋一句可行動；每日抽 4 則（日期種子，同日固定）
更新update-ticker.ps1 → ticker.json／ticker.html
桌面檔只留 今日實作-YYYY-MM-DD.txt；換日刪前一天；不產無日期檔
重開雙擊 今日實作跑馬燈.lnk；若已開則叫到最前；僵死後會清掉再啟

學到的技術點

  • 瀏覽器 app 模式不如 WinForms 穩定「貼在桌面」；置頂＋工作區底部更可靠。
  • PowerShell 腳本含中文時需 UTF-8 BOM；捷徑宜指向 ASCII 路徑的 .cmd。
  • 單實例用 Mutex：再開時應「叫到最前」，不要只跳訊息卻看不到。
  • 摘取要點：一句一事、今晚可做，勝過長篇勵志。

四｜今日行程跑馬燈設計

路徑：MyKnowledgeBase\今日行程\。沿用桌面跑馬燈的操作概念，改成螢幕頂部的暮金色行程列，與底部「今日實作」同時存在。

項目設計與做法
資料來源讀取 Google 日曆私密 iCal；私密網址只保存在本機設定，不寫入公開日誌或 GitHub
同步每 15 分鐘更新今日行程；開始前 30 分鐘標示「即將」
視覺螢幕頂部置頂、暮金色系；左側顯示「今日行程」與日期，中央勻速捲動行程
操作右上 X 關閉；其下「日」按鈕直接開啟 Google 日曆；Esc 亦可關閉
安全開啟前輸入本機密碼；僅保存 PBKDF2 雜湊，不保存明文；可重設密碼
桌面檔只保留 今日行程-YYYY-MM-DD.txt，換日自動移除前一天
排錯修正舊週期事件 COUNT 誤判，避免過期活動被列入今日；修正「日」字按鈕裁切
最終形成「上方看行程、下方看實作」的雙跑馬燈：一條提醒今天要去哪裡，一條提醒今天要實踐什麼。

本次技術重點

  • iCal 週期事件需正確處理 COUNT／UNTIL，不能把多年以前的活動延伸到今天。
  • 中文小字在 WinForms Button 可能被 GDI 裁切；改用固定上下兩格與自繪字形較可靠。
  • 公開紀錄只寫方法，不寫私密 iCal 網址、密碼雜湊或實際私人行程。
